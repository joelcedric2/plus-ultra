# PlusUltra

Web frontend for PlusUltra, an LLM council platform that lets you run prompts through multiple AI models simultaneously and compare their responses side by side.

Built with React, TypeScript, Vite, and shadcn/ui.

## Features

- Workspace for composing and submitting prompts to multiple LLMs
- Side-by-side response comparison
- User accounts with sign-in/sign-up
- Publishing and sharing workflows

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
